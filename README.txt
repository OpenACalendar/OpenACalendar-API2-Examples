OpenACalendar - API2 Examples
=============================

See http://ican.openacalendar.org/ for full information and all documentation.


CLI
===

Command Line scripts to demonstrate editing.

Copy config.php.dist to config.php and edit.

Run auth.php to get a user token and secret, and edit these into your config.php.

To demonstrate the API, run: 
  *  group.php
  *  event.php
  *  area.php
  *  venue.php
